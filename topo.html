Para onde conectam os links no topo de todas páginas?



Sala Estudos: links para os autores dos textos, para os nikayas (as coleções dos suttas), para os guias de estudo e glossários.

Sutta Pitaka: descrição e os links para os nikayas, as coleções dos suttas.

Índice Suttas: todos os suttas em ordem alfabética, com uma breve descrição de cada um.

Índice Textos: todos os textos e guias de estudo em ordem alfabética, com uma breve descrição de cada um.

Livros: Todos os livros publicados pelo site.

Audiobook: Alguns dos livros também estão disponíveis no formato audiobook compatível com Apple (M4B) ou em MP3.

Podcast: áudios contendo estudos de suttas e estudos temáticos.

Ajuda: perguntas mais frequentes e dicas sobre como navegar no site.

DHP Random: versos aleatórios do Dhammapada. Pode ser utilizado para uma leitura do tipo "um verso por dia".

Sutta Random: suttas aleatórios do Itivuttaka. São suttas muito curtos que podem ser utilizados para uma leitura do tipo "um sutta por dia".
