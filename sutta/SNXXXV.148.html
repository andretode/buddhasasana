"Bhikkhus, eu lhes ensinarei a prática que é adequada para realizar nibbana. Ouçam e prestem muita atenção ...

"Bhikkhus, qual é a prática adequada para realizar nibbana? Aqui, um bhikkhu vê o olho como sofrimento, ele vê as formas como sofrimento, ele vê a consciência no olho como sofrimento, ele vê o contato no olho como sofrimento, ele vê como sofrimento qualquer sensação que surja tendo o contato no olho como condição, quer seja prazerosa, dolorosa, ou nem prazerosa bem dolorosa.

"Ele vê o ouvido como sofrimento ... Ele vê a mente como sofrimento, ele vê os objetos mentais como sofrimento, ele vê a consciência na mente como sofrimento, ele vê o contato na mente como sofrimento, ele vê como sofrimento qualquer sensação que surja tendo o contato na mente como condição, quer seja prazerosa, dolorosa, ou nem prazerosa bem dolorosa.

"Essa bhikkhus é a prática que é adequada para realizar nibbana."
