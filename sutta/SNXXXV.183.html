Samyutta Nikaya XXXV.183-185

Anattachanda Sutta

Desejo pelo Não-eu

Somente para distribuição gratuita.
Este trabalho pode ser impresso para distribuição gratuita.
Este trabalho pode ser re-formatado e distribuído para uso em computadores e redes de computadores
contanto que nenhum custo seja cobrado pela distribuição ou uso.
De outra forma todos os direitos estão reservados.

 

"Bhikkhus, vocês devem abandonar o desejo por tudo que é não-eu ... vocês devem abandonar a cobiça por tudo que é não-eu ... vocês devem abandonar o desejo e a cobiça por tudo que é não-eu. O que é não-eu? As formas são não-eu ... os objetos mentais são não-eu; vocês devem abandonar o desejo e a cobiça por isso. Bhikkhus, vocês devem abandonar o desejo e a cobiça por tudo que é não-eu."
